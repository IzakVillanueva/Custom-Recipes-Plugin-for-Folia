# Custom-Recipes-Plugin-for-Folia

This is a lightweight and scalable custom recipes plugin made for Folia used in Ironclad Network.

It includes reverse crafting a block into its item variant.

Current recipe list:
1 Quartz Block -> 4 Quartz
1 Prismarine -> 4 Prismarine Shard
1 Prismarine Bricks -> 9 Prismarine Shard
1 Honeycomb Block -> 4 Honeycomb
1 Brick -> 4 Bricks
Default Bow Recipe with Dropper in the middle -> 1 Dispenser
